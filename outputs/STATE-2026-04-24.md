## Current Invariants (DO NOT REGRESS)

### Audio
- **OYE never stops**: `bgEngine.ts` — if track ends and autoplay fails, bgEngine watchdog retries
- **Silence bridge**: `playbackState.ts` ALLOWED includes `idle→bridge` — without this, natural track ends log `state_illegal` and silent WAV is rejected, breaking autoplay
- **QueueItem shape**: `{ track: Track; addedAt: string; source: string }` — always access `.track.title`, `.track.trackId`, etc.
- **State machine**: `playbackState.ts` guards transitions. `state_illegal` in telemetry = real bug.

### VoyoLiveCard (SignInPrompt.tsx)
- **Mode machine**: `community → dissolve (350ms) → next (orange) → [7s] → dissolve → community`
- **Progress triggers**: `[0.07, 0.38, 0.65, 0.86]` — `shownCountRef` increments each time, reset on track change
- **Live dot**: orange `#F97316` when `isPlaying`, green `#2DB785` when paused
- **"N in Voyo"**: only shows when `liveCount > 0` (real friends playing)
- **Next track**: from `queue[0]` (QueueItem) — render as `nextTrack.track.title` etc.

### Classics (HomeFeed.tsx)
- **Shimmer**: `classics-subtitle-shimmer` keyframe + random `animationDuration` via `shimmerDuration` ref (computed once on mount, `18–28s` range)
- **Disc glow+spin**: `isSelected={selectedClassic?.id === track.id}` drives the CSS state
- **RR Roof fades**: left/right absolute overlays inside `relative overflow-hidden` wrapper

- **MUST** bump `public/version.json` (`YYYY.MM.DD.NNN`, `force:true`) for UpdateButton to trigger client reload

## Telemetry
